Build a Calculator in JavaScript

Step 1:What It Should Do?

The calculator should:
    Ask the user for two numbers Ask for an operator (`+`, `-`, `*`, `/`) Perform the calculation Show the result
Step 2: Write the Code

Simple Calculator in JavaScript

Get user input

    let num1 = parseFloat(prompt("Enter the first number:")); let op = prompt("Enter an operator (+, -, *, /):"); let num2 = parseFloat(prompt("Enter the second number:"));
let result;

Perform the calculation:

if (op === "+") {
result = num1 + num2;
} else if (op === "-") {
result = num1 - num2;
} else if (op === "*") {
result = num1 * num2;
} else if (op === "/") {
if (num2 !== 0) {
result = num1 / num2;
} else {
result = "Error: Cannot divide by zero";
}
} else {
result = "Invalid operator";
}

Show the result:

alert("Result: " + result);

Step 3: Upgrades You Can Try
  1. Looping: Let the user calculate repeatedly until they exit.
  2. Functions: Wrap operations in functions like `add(a, b)`.
  3. GUI Version: Create HTML buttons and display for a real calculator interface.
