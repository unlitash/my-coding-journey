build a simple calculator in Python


Step 1: Understand What You Want

The calculator should:
  1. Ask the user to input two numbers
  2. Ask the user to choose an operation: `+`, `-`, `*`, `/`
  3. Perform the calculation
  4. Show the result
Step 2: Understand What You Want Write the Code

Simple Calculator in Python:

Ask for user input:

  1. num1 = float(input("Enter the first number: "))
  2. op = input("Enter an operator (+, -, *, /): ")
  3. num2 = float(input("Enter the second number: "))

Perform the calculation:


if op == '+':
result = num1 + num2
elif op == '-':
result = num1 - num2
elif op == '*':
result = num1 * num2
elif op == '/':
if num2 != 0:
result = num1 / num2
else:
result = "Error: Cannot divide by zero"
else:
result = "Invalid operator"

Print the result:

print("Result:", result)

Example Output:

Enter the first number: 10 Enter an operator (+, -, *, /): * Enter the second number: 5 Result: 50.0

Want to make it cooler? You can:
